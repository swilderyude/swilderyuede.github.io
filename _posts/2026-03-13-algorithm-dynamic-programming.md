# 算法系列：动态规划从入门到精通

## 1. 动态规划概述

动态规划（Dynamic Programming，简称 DP）是一种解决具有重叠子问题和最优子结构性质问题的方法。它通过将原问题分解为子问题，存储子问题的解，避免重复计算，从而提高算法效率。

### 1.1 核心思想

- **重叠子问题**：原问题可以分解为若干个子问题，这些子问题会被重复计算
- **最优子结构**：原问题的最优解包含子问题的最优解
- **状态转移**：通过状态转移方程，从子问题的解推导出原问题的解

### 1.2 适用场景

- 最优化问题
- 计数问题
- 存在性问题

## 2. 动态规划的基本步骤

### 2.1 定义状态

状态是描述问题当前情况的一组变量，通常用 `dp[i]` 或 `dp[i][j]` 表示。

### 2.2 确定状态转移方程

状态转移方程描述了如何从子问题的解得到当前问题的解。

### 2.3 初始化

初始化边界条件，如 `dp[0]`、`dp[0][0]` 等。

### 2.4 计算顺序

按照依赖关系，从小到大计算状态。

### 2.5 输出结果

根据最终状态得到问题的解。

## 3. 经典例题解析

### 3.1 斐波那契数列

**问题描述**：求第 n 个斐波那契数。

**状态定义**：`dp[i]` 表示第 i 个斐波那契数。

**状态转移方程**：`dp[i] = dp[i-1] + dp[i-2]`

**初始化**：`dp[0] = 0`, `dp[1] = 1`

**代码实现**：

```python
def fib(n):
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
```

**优化空间**：可以使用滚动数组减少空间复杂度到 O(1)。

```python
def fib(n):
    if n <= 1:
        return n
    a, b = 0, 1
    for i in range(2, n + 1):
        a, b = b, a + b
    return b
```

### 3.2 爬楼梯

**问题描述**：有 n 阶楼梯，每次可以爬 1 或 2 阶，求有多少种不同的爬法。

**状态定义**：`dp[i]` 表示爬到第 i 阶的方法数。

**状态转移方程**：`dp[i] = dp[i-1] + dp[i-2]`

**初始化**：`dp[1] = 1`, `dp[2] = 2`

**代码实现**：

```python
def climbStairs(n):
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    dp[2] = 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
```

### 3.3 最长递增子序列

**问题描述**：给定一个无序的整数数组，找到其中最长递增子序列的长度。

**状态定义**：`dp[i]` 表示以第 i 个元素结尾的最长递增子序列的长度。

**状态转移方程**：`dp[i] = max(dp[j] + 1) for j < i and nums[j] < nums[i]`

**初始化**：`dp[i] = 1`（每个元素自身构成一个子序列）

**代码实现**：

```python
def lengthOfLIS(nums):
    if not nums:
        return 0
    n = len(nums)
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

**优化**：使用贪心 + 二分查找可以将时间复杂度降低到 O(n log n)。

### 3.4 0-1 背包问题

**问题描述**：给定 n 个物品，每个物品有重量 w[i] 和价值 v[i]，以及一个容量为 C 的背包，求能装入背包的最大价值。

**状态定义**：`dp[i][j]` 表示前 i 个物品，容量为 j 时的最大价值。

**状态转移方程**：
- 不选第 i 个物品：`dp[i][j] = dp[i-1][j]`
- 选第 i 个物品：`dp[i][j] = dp[i-1][j-w[i]] + v[i]`
- 取两者最大值：`dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i])`

**初始化**：`dp[0][j] = 0`（前 0 个物品，价值为 0）

**代码实现**：

```python
def knapsack(w, v, C):
    n = len(w)
    dp = [[0] * (C + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for j in range(1, C + 1):
            if j >= w[i-1]:
                dp[i][j] = max(dp[i-1][j], dp[i-1][j - w[i-1]] + v[i-1])
            else:
                dp[i][j] = dp[i-1][j]
    return dp[n][C]
```

**优化空间**：可以使用一维数组减少空间复杂度到 O(C)。

```python
def knapsack(w, v, C):
    n = len(w)
    dp = [0] * (C + 1)
    for i in range(n):
        for j in range(C, w[i]-1, -1):
            dp[j] = max(dp[j], dp[j - w[i]] + v[i])
    return dp[C]
```

## 4. 高级动态规划技巧

### 4.1 状态压缩

对于某些问题，状态可以用二进制表示，从而减少状态空间。

**示例**：旅行商问题（TSP）

**状态定义**：`dp[mask][u]` 表示访问过的城市集合为 mask，当前在城市 u 时的最小距离。

**状态转移方程**：`dp[mask | (1 << v)][v] = min(dp[mask | (1 << v)][v], dp[mask][u] + dist[u][v])`

### 4.2 数位动态规划

用于处理与数字相关的计数问题。

**示例**：计算 [1, n] 中不含数字 4 的数的个数。

**状态定义**：`dp[pos][tight][has4]` 表示处理到第 pos 位，是否受限于原数，是否已经包含数字 4 的状态。

### 4.3 树形动态规划

在树结构上进行动态规划。

**示例**：树的最大独立集

**状态定义**：`dp[u][0]` 表示不选节点 u 时的最大权值和，`dp[u][1]` 表示选节点 u 时的最大权值和。

**状态转移方程**：
- `dp[u][0] = sum(max(dp[v][0], dp[v][1])) for v in children(u)`
- `dp[u][1] = val[u] + sum(dp[v][0]) for v in children(u)`

## 5. 动态规划的常见陷阱

### 5.1 状态定义不清晰

状态定义是动态规划的核心，需要确保状态能够准确描述问题的当前情况。

### 5.2 状态转移方程错误

需要仔细分析问题，确保状态转移方程的正确性。

### 5.3 初始化错误

边界条件的初始化直接影响最终结果，需要仔细考虑。

### 5.4 计算顺序错误

状态的计算顺序需要根据依赖关系确定，确保在计算当前状态时，所依赖的子状态已经计算完成。

## 6. 总结

动态规划是一种强大的算法设计方法，掌握它需要大量的练习和思考。通过分析经典问题，理解动态规划的核心思想和解题步骤，你可以逐渐提高解决复杂问题的能力。

**练习建议**：
- 从简单问题开始，如斐波那契数列、爬楼梯等
- 逐步挑战中等难度的问题，如最长递增子序列、0-1 背包等
- 尝试解决高级问题，如状态压缩 DP、数位 DP、树形 DP 等
- 总结不同类型问题的解题模式和技巧

**记住**：动态规划的关键在于找到正确的状态定义和状态转移方程，这需要对问题的深入理解和大量的实践。
