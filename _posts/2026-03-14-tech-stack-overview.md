# 技术栈 overview：计算机研究生必备技能

## 1. 编程语言

### 1.1 Python

**应用领域**：机器学习、数据分析、Web 开发

**核心库**：
- 数据科学：NumPy, Pandas, Matplotlib, Seaborn
- 机器学习：scikit-learn, PyTorch, TensorFlow
- Web 开发：Flask, Django, FastAPI
- 自动化：Selenium, BeautifulSoup

**学习资源**：
- 《Python 编程：从入门到实践》
- [Python 官方文档](https://docs.python.org/3/)
- [Real Python](https://realpython.com/)

### 1.2 C++

**应用领域**：高性能计算、系统编程、游戏开发

**核心库**：
- STL (Standard Template Library)
- Boost
- OpenCV (计算机视觉)
- Eigen (线性代数)

**学习资源**：
- 《C++ Primer》
- [C++ 参考手册](https://en.cppreference.com/w/)
- [C++ Core Guidelines](https://github.com/isocpp/CppCoreGuidelines)

### 1.3 Java

**应用领域**：企业级应用、Android 开发、大数据

**核心框架**：
- Spring Boot
- Hibernate
- Apache Spark
- Android SDK

**学习资源**：
- 《Effective Java》
- [Oracle Java 教程](https://docs.oracle.com/javase/tutorial/)
- [Spring 官方文档](https://spring.io/docs)

## 2. 前端技术

### 2.1 HTML/CSS/JavaScript

**核心技术**：
- HTML5 (语义化标签、Canvas、SVG)
- CSS3 (Flexbox、Grid、动画)
- JavaScript (ES6+、异步编程、DOM 操作)

**学习资源**：
- [MDN Web Docs](https://developer.mozilla.org/zh-CN/)
- [W3Schools](https://www.w3schools.com/)
- 《JavaScript 高级程序设计》

### 2.2 前端框架

**主流框架**：
- React (Facebook)
- Vue.js (尤雨溪)
- Angular (Google)

**辅助工具**：
- Webpack (打包工具)
- Babel (ES6+ 转译)
- ESLint (代码规范)
- Prettier (代码格式化)

**学习资源**：
- [React 官方文档](https://reactjs.org/docs/getting-started.html)
- [Vue.js 官方文档](https://vuejs.org/)
- [Angular 官方文档](https://angular.io/docs)

## 3. 后端技术

### 3.1 服务器端框架

**Python**：
- Flask (轻量级)
- Django (全功能)
- FastAPI (高性能)

**Node.js**：
- Express
- NestJS
- Koa

**Java**：
- Spring Boot
- Quarkus
- Micronaut

### 3.2 数据库

**关系型数据库**：
- MySQL
- PostgreSQL
- SQLite

**NoSQL 数据库**：
- MongoDB (文档型)
- Redis (键值型)
- Cassandra (列族型)

**学习资源**：
- 《SQL 必知必会》
- [PostgreSQL 官方文档](https://www.postgresql.org/docs/)
- [MongoDB 官方文档](https://docs.mongodb.com/)

## 4. 机器学习与人工智能

### 4.1 核心库

- **PyTorch**：动态计算图，适合研究和原型开发
- **TensorFlow**：静态计算图，适合生产部署
- **scikit-learn**：传统机器学习算法
- **XGBoost**：梯度提升树算法
- **LightGBM**：高效梯度提升框架

### 4.2 深度学习框架

- **Keras**：高级神经网络 API
- **PyTorch Lightning**：PyTorch 的高级封装
- **FastAI**：快速 AI 开发

### 4.3 学习资源

- 《深度学习》(花书)
- 《机器学习》(周志华)
- [吴恩达机器学习课程](https://www.coursera.org/learn/machine-learning)
- [Stanford CS229](https://cs229.stanford.edu/)

## 5. 开发工具与环境

### 5.1 版本控制

- **Git**：分布式版本控制系统
- **GitHub/GitLab**：代码托管平台
- **Git Flow**：分支管理策略

**学习资源**：
- [Pro Git](https://git-scm.com/book/zh/v2)
- [GitHub 官方文档](https://docs.github.com/)

### 5.2 集成开发环境 (IDE)

- **Visual Studio Code**：轻量级、插件丰富
- **PyCharm**：Python 专用 IDE
- **IntelliJ IDEA**：Java 专用 IDE
- **VS Code**：通用代码编辑器

### 5.3 容器与虚拟化

- **Docker**：容器化技术
- **Kubernetes**：容器编排
- **Docker Compose**：多容器应用管理

**学习资源**：
- [Docker 官方文档](https://docs.docker.com/)
- [Kubernetes 官方文档](https://kubernetes.io/docs/)

## 6. 系统与网络

### 6.1 操作系统

- **Linux**：Ubuntu、CentOS、Debian
- **Shell 脚本**：Bash、Zsh
- **系统工具**：top、htop、ps、netstat

**学习资源**：
- 《鸟哥的 Linux 私房菜》
- [Linux 命令大全](https://man.linuxde.net/)

### 6.2 网络编程

- **HTTP/HTTPS**：网络协议
- **RESTful API**：API 设计规范
- **WebSocket**：实时通信
- **gRPC**：高性能 RPC 框架

**学习资源**：
- 《计算机网络：自顶向下方法》
- [MDN Web API 文档](https://developer.mozilla.org/zh-CN/docs/Web/API)

## 7. 软技能

### 7.1 文档编写

- **Markdown**：轻量级标记语言
- **LaTeX**：学术论文排版
- **技术文档**：API 文档、架构设计文档

### 7.2 项目管理

- **敏捷开发**：Scrum、Kanban
- **项目管理工具**：Jira、Trello、GitHub Projects
- **代码审查**：Pull Request、Code Review

### 7.3 沟通与协作

- **技术演讲**：PPT 制作、演讲技巧
- **团队协作**：Git 协作、代码规范
- **技术写作**：博客、技术分享

## 8. 总结

作为计算机研究生，掌握这些技术栈将有助于你在科研和项目中取得更好的成果。技术发展迅速，建议保持学习的热情，关注最新的技术动态，并根据自己的研究方向和兴趣进行深入学习。

**持续学习，不断进步！**
